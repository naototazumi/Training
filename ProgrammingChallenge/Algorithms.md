# 除算の切り上げ、切り下げ
C++：正の整数を正の整数で割った場合、答えは必ず切り捨てになる。
Python：切り捨て除算//は除算値を超えない最大の整数
```
C++
# 整数A,BについてA/Bの切り捨てを求める書き方
F = A / B
# 整数A,BについてA/Bの切り上げを求める書き方
C = (A + B - 1) / B

Python
# 整数A,BについてA/Bの切り捨てを求める書き方
F = A // B
# 整数A,BについてA/Bの切り上げを求める書き方
C = -(-A // B)
```
